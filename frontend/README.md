#Cách chạy project
B1: clone project từ github https://github.com/Huywb/MGITest.git
B2: in Terminal enter npm i to download libary in here i use Axios so it need to be install
B3: npm run dev to run project

#Các chức năng đã hoàn thành 
 +: Fetch API danh sách products và show ra màn hình
 +: Responsive ở cả desktop (8 sản phẩm) tablet (3 sản phẩm) mobile (2 sản phẩm)
 +: Tạo ô tìm kiếm đề người dùng nhập tên sản phẩm (Không phân biệt chữ Hoa và có hiển thị thông báo nếu không tìm  được sản phẩm)
 +: Tạo popup hiển thị chi tiết sản phẩm (tên, mô tả, giá, loại sản phẩm) có thể click bên ngoài popup để đóng popup
 +: Tạo loading khi fetch sản phẩm 
 +: Các hiệu ứng hover loading effect
 +: Phân trang với mỗi trang 8 sp
 +: Tạo bộ lọc theo giá (Tăng dần, giảm dần)

#Các khó khăn gặp phải
